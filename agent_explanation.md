# 🤖 AI Agents

## 🔹 What are AI Agents?
AI Agents are LLM systems that can:
- Think
- Decide actions
- Use external tools

---

## 🔹 Examples of Tools:
- Search engine
- Calculator
- Database retrieval
- APIs

---

## 🔹 Workflow:
1. User gives input  
2. LLM decides action  
3. Tool is used  
4. Result is processed  
5. Final response is generated  

---

## 🔹 Key Difference:
- RAG → Retrieves information  
- Agents → Take actions
