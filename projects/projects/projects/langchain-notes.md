
# 🧠 LangChain Notes

## 📌 Overview
LangChain is a framework used to build applications powered by Large Language Models (LLMs).

---

## ⚙️ Key Components
- Chains (sequence of LLM calls)
- Agents (decision-making systems)
- Tools (APIs, search, calculators)
- Memory (conversation history)

---

## 🧩 Why LangChain is useful
- Simplifies LLM app development
- Helps build AI agents easily
- Integrates with vector databases

---

## 🚀 Use Cases
- Chatbots
- AI assistants
- Document Q&A systems (RAG)
- Automation workflows

---

## 📚 Learning Outcome
I understood how developers build structured LLM applications instead of just prompting models.
