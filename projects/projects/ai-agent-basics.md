# 🤖 AI Agent Basics

## 📌 Overview
AI Agents are systems that can think, plan, and take actions using tools to complete tasks autonomously.

---

## 🧠 How AI Agents Work
1. Receive task from user
2. Plan steps to solve it
3. Use tools (APIs, search, calculators, etc.)
4. Observe results
5. Repeat until task is completed

---

## ⚙️ Core Concepts
- Agent Loop (Think → Act → Observe)
- Tool usage
- Memory (short-term / long-term)
- Planning and reasoning

---

## 🧩 Example Use Case
Task: "Book me a flight"

Agent:
- Searches flights
- Compares prices
- Selects best option
- Returns result

---

## 🚀 Why Agents matter
- Automate complex workflows
- Power tools like AutoGPT, LangGraph systems
- Used in real enterprise automation

---
