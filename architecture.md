# 🏗 AI System Architecture

## 🔹 RAG (Retrieval-Augmented Generation)
RAG improves LLM responses by using external knowledge sources.

### Flow:
1. User asks a question  
2. Query is converted into embeddings  
3. Vector database finds similar chunks  
4. Relevant context is retrieved  
5. LLM generates final answer using context  

---

## 🔹 AI Agents
AI Agents are LLM systems that can:
- Use tools
- Make decisions
- Perform multi-step reasoning

### Flow:
User → LLM → Tool Selection → Execution → Final Response

---

## 📌 Key Idea
- RAG = Knowledge-based AI
- Agents = Decision-making AI
